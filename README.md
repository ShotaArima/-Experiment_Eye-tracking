# Experiment_Eye-tracking

## 課題
- データの読み取りに際して、しばしば誤読することやそもそも読み方が、暗黙知的な側面が考えられます。
- そこで、本実験では、表やデータの読み取りについての正答率や時間、視線の動きをもとに考察し、研究の見せ方についての応用を目的とする
